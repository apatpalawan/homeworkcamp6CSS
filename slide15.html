value ใน display property มีอะไรบ้าง

1.None ไม่แสดงผล
2.block block
3.inline-block แสดงผลแบบ inline block

grid และ flex แตกต่างกันอย่างไร

grid คือ การแบ่งพื้นที่ออกเป็นส่วน ๆ พื้นที่หน้าจอจะถูกแบ่งตามเส้นจากบนลงล่าง และจากซ้ายไปขวา flex คือ การออกแบบพื้นที่โดยการจัดเรียงไปแนวนอน หรือแนวตั้ง สามารถย้าย item ต่าง ๆ ได้

justify and align แตกต่างกันอย่างไร

align จะเป็นการจัดบรรทัดของข้อความจากขอบซ้ายไปจนถึงขอบด้านขวา จะไม่สามารถขยายขอบได้ justify จะทำการจัดข้อความโดยสามารถขยายกรอบของข้อความได้

position z-index

position คือ ตำแหน่งการจัดวางของ css ประกอบไปด้วย

  1. static
  2. relative
  3. absolute
  4. fixed

z-index คือ ใช้จัดเรียง element ที่ซ้อนกัน โดยจะมีผลก็ต่อเมื่อมีการใส่ positon ให้กับ element นั้น ๆ
โดยจะจัดเรียงตามค่าตัวเลขที่กำหนด ตั้งแต่เลขบวก จนติดลบ
